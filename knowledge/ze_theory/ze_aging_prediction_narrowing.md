## Clinical Summary for AIM

Ze aging is the progressive elevation of the T-event threshold in a biological system.
The system increasingly ignores signals that don't match its existing model.

**Diagnostic consequence:** Aging patients present with:
- Reduced HRV (Ze: narrow RR = low T-sensitivity)
- Slower EEG alpha peak (Ze: v drifts from v*)
- Confirmation bias in symptoms (Ze: rounding off weak T-signals)
- Delayed immune response (Ze: requires stronger antigen signal to register T)

**AIM application:** Low χ_Ze in HRV + slow α-peak = Ze-aging signature.
Use Ze age (v distance from v*) alongside chronological age in assessment.

## Mechanism (brief)

1. Ze predicts S (norm) before each physiological event
2. T-event (deviation from norm) = informational stress
3. Young system: updates model → stays at v*
4. Aging system: raises threshold → rounds off T → v drifts → complexity falls
5. Limit: θ → ∞, τ → 0, v → -1 (all S, no surprise, death)

## Therapeutic Protocol Implication

Hormesis interventions work by **forcing sub-threshold T-events**:
- Cold, fasting, exercise, novelty, social unpredictability
- Goal: prevent θ from rising monotonically
- Target: maintain v ≈ v* = 0.456

## Ze-Gödel Link

The system cannot detect that its model is outdated
because it uses the same outdated model to evaluate new signals.
Threshold drift = functional incompleteness.

## Wave Function / Measurement

Ze prediction = choice of measurement basis.
Event arrival = wave function collapse (T or S).
Coarse prediction = wide S-zone = subtle T-waves become S-corpuscles.
**Aging Ze creates false negatives by measurement, not by absence of signal.**
