## ✅ ЖЕЛАТЕЛЬНО (разрешено, рекомендуется)

### Хлеб и зерновые
- Хлеб и выпечка на **винных дрожжах**
- Пшеница (зерно, пшеничная каша / ხორბლის ბურღული)
- Просо / ფეტვი
- Перловая крупа / ქერი

### Яйца
- Яйца **сваренные вкрутую минимум 40 минут** от закипания
- Яйца **хорошо прожаренные с обеих сторон**

### Мясо и птица
- Говядина обескровленная (первый бульон слить)
- Курица без кожи (первый бульон слить)
- Баранина обескровленная (первый бульон слить)
- Индейка (без кожи, первый бульон слить)

### Рыба
- Любая рыба (жареная с ткемали, варёная в кинзмари и др.)

### Молочные продукты
- Имеретинский сыр (рассольный)
- Сулугуни (рассольный)
- Сливочное масло / კარაქი
- Топлёное масло (ерба) / ერბო

### Масла
- Рафинированное подсолнечное масло

### Орехи и семена
- Грецкий орех / ნიგოზი
- Ореховый баже на воде
- Миндаль (умеренно) / ნუში
- Тыквенные семечки / გოგრის თესლი

### Бобовые
- Фасоль (лобио) / ლობიო
- Зелёная фасоль / მწვანე ლობიო

### Овощи
- Лук репчатый / ხახვი
- Красный лук / წითელი ხახვი
- Лук-порей / პრასი
- Морковь / სტაფილო
- Свёкла / ჭარხალი
- Грибы / სოკო
- Огурец / კიტრი
- Тыква / გოგრა
- Кабачок / ყაბაყი
- Болгарский перец / ბულგარული წიწაკა
- Полевое пхали / მინდვრის ფხალი
- Авокадо / ავოკადო
- Оливки / ზეთისხილი

### Квашения и ферментированное
- Джонджоли (квашеный) / ჯონჯოლი
- Квашеный огурец / მჟავე კიტრი
- Квашеный чеснок / მჟავე ნიორი
- Ткемали без перца и сахара (с варёным чесноком) / ტყემალი

### Специи и зелень
- Укроп / კამა
- Сельдерей / ნიახური
- Кинза / ქინძი
- Базилик / რეჰანი
- Шафран / ზაფრანა
- Чабер (кондари) / ქონდარი
- Сушёные специи / ხმელი სუნელი
- Тмин / ძირა
- Коцахури / Барбарис / კოწახური → перемещён в раздел УМЕРЕННО

### Фрукты (только в 1-й половине дня)
- Яблоко / ვაშლი
- Груша / მსხალი
- Персик / ატამი
- Слива (все сорта) / ქლიავი
- Айва / კომში
- Малина / ჟოლო
- Черника / მოცვი
- Ежевика / მაყვალი
- Гранат / ბროწეული
- Кизил / შინდი
- Вишня (кислая) / ალუბალი
- Клубника (умеренно) / მარწყვი
- Варенья и сухофрукты из вышеперечисленного (умеренно)

### Напитки и прочее
- Вода (за 5–30 минут до еды, не менее 150 мл)
- Кофе без сахара (до еды или через 2 часа после, не сразу после еды)
- Настой липы (через 2 часа после еды)
- Настой черники (через 2 часа после еды)
- Настой малины (через 2 часа после еды)
- Настой чабера (через 2 часа после еды)
- Настой шафрана / ზაფრანას ნაყენი
- Детокс настой
- Сахар: 1–3 чайных ложки в день максимум

- Рис коричневый / ყავისფერი ბრინჯი

- Брокколи / ბროკოლი
- Спагетти / паста (пшеничная) / სპაგეტი
- Миндаль / ნუში
- Клубника / მარწყვი
- Кролик / კურდღელი (первый бульон слить)
- Вода Ликани / Набеглави (иногда)
