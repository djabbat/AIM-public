# PARAMETERS.md — конфигурируемые значения SSA-движка

Версия: AIM-SSA v0.1  
Дата: 2025-03-27  
Основание: ICSH 2014, NHANES, BCSH, WHO  

## 1. Engine

| Параметр | Значение | Обоснование / Ссылка |
|---|---|---|
| `ZONE_BORDER_TOLERANCE_PCT` | 2.0 | Граница зоны ±2% — флаг «borderline», компенсация аналитической вариации (ICSH 2014, Westgard) |
| `TOP_K_SYNDROMES` | 10 | Максимум синдромов в ранжированном списке, выдаваемом в DiffDiagnosis |
| `MIN_PATTERN_SPECIFICITY` | 0.3 | Минимальная специфичность паттерна (1 – false positive rate) для включения в ранжирование |
| `ENGINE_TIMEOUT_MS` | 1500 | Тайм-аут на один запрос к движку, мс (обеспечение отклика <2 с) |

## 2. API (Axum)

| Параметр | Значение | Примечание |
|---|---|---|
| `API_PORT` | 8766 | DiffDiagnosis занимает 8765; SSA — смежный порт |
| `API_RATE_LIMIT_RPM` | 60 | Запросов в минуту на IP; защита от DDoS |

## 3. UI (Phoenix LiveView)

| Параметр | Значение | Примечание |
|---|---|---|
| `PHOENIX_PORT` | 4001 | DiffDiagnosis: 4000; SSA UI — 4001 |

## 5. Метрики качества (целевые)

| Метрика | Target | Метод валидации |
|---|---|---|
| `top-1 syndrome accuracy` | ≥ 0.70 | Gold-set n=200 (верифицированные диагнозы, 5 экспертов) |
| `top-3 syndrome accuracy` | ≥ 0.90 | Совпадение хотя бы одного синдрома из top-3 с диагнозом |
| `red-flag-miss-rate` | ≤ 0.02 | Пропуск панцитопении, ТМА, лейкоэритробластной картины — недопустимо |
| `calibration (ECE)` | < 0.05 | Expected Calibration Error (DeGroot 1983, Guo 2017) |
| `time-to-pattern` | < 1.5 с | Время от получения CBC-вектора до ранжированного списка |
